# 🐱 CatEscape (고양이 피하기 게임)

**CatEscape**는 유니티 엔진(2022.3.62f3)을 활용하여 제작한 2D 게임입니다.
하늘에서 무작위로 떨어지는 화살표 비를 피해 고양이를 안전하게 보호해보세요!

---

## 🛠 개발 환경 (Environment)
*   **Unity Version:** 2022.3.62f3 (LTS)
*   **Language:** C#
*   **Platform:** PC (Windows)

---

## 🎮 게임 조작법 (Controls)
키보드의 방향키를 사용하여 고양이를 좌우로 이동시킬 수 있습니다.

| 키 (Key) | 동작 (Action) |
| :--- | :--- |
| **Left Arrow (←)** | 왼쪽으로 이동 |
| **Right Arrow (→)** | 오른쪽으로 이동 |
